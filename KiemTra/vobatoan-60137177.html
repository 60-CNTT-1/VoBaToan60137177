What is Ajax?


As defined in Wikipedia



AIAX is a group of interrelated web development techniques used to create interactive web applications or rich internet applications. With AJax, web applications can retrieve data from the server asynchronously in the background without interfering with the display and behavior of the existing page


Brief History

Year Event
1995 Java applets were introduced-the firt way to load data from client side asynchronously
1996 Iframe element were introduced, which also have this functionally
1999 Microsoft created the XMLHTTP ActiveX control in Internet EXplorer 5 which then followed by other browser as XMLHttpRequest object
2005 The term "Ajax" was borned
2006 The W3C in order to to make it offical web standard, publicized the first draft specification for XMLHttpRequest Object
